Действительно ли OpenOffice бесплатен для пользователей?

OpenOffice бесплатен для всех. Вы можете взять эту копию OpenOffice и установить её на любое количество компьютеров, преследуя любые цели (включая коммерческие, государственные, образовательные). Подробности описаны в лицензии, поставляемой вместе с OpenOffice, или по адресу http://www.openoffice.org/license.html

Почему OpenOffice бесплатен?

Вы можете использовать данную копию OpenOffice бесплатно потому, что независимые разработчики и корпоративные спонсоры спроектировали, разработали, оттестировали, перевели, документировали, занимались распространением и поддержкой, а также многими другими способами помогли OpenOffice стать тем, чем он является сегодня, - мировым лидером среди офисных приложений в области свободного программного обеспечения.

Если вы цените их усилия и хотите помочь обеспечить проекту Apache OpenOffice будущее, подумайте о возможности поддержать его, подробнее об участии в разработке на http://openoffice.apache.org/get-involved.html, о пожертвованиях на http://www.apache.org/foundation/contributing.html. Каждый способен помочь.

Замечания по программе установки

Для полной функциональности OpenOffice требует свежую версию JAVA, которую возможно загрузить на http://java.com.

Системные требования:

  • Microsoft Windows XP, Vista, Windows 7 или Windows 8

  • Процессор Pentium III или лучше

  • 256 МБ ОЗУ (рекомендуется 512 МБ ОЗУ)

  • До 1.5 ГБ свободного пространства на жестком диске

  • Разрешение 1024 x 768 и 256 цвета (рекомендуется большее разрешение)

Примечание: для выполнения процесса установки необходимы права администратора.

Разрешить или запретить регистрацию OpenOffice как основного приложения для форматов Microsoft Office можно при помощи следующих параметров командной строки программы установки:

  • /msoreg=1 разрешить регистрацию OpenOffice как основного приложения для форматов Microsoft Office.

  • /msoreg=0 запретить регистрацию OpenOffice как основного приложения для форматов Microsoft Office.

Если выполняете административную установку с помощью команды setup /a, следует убедиться, что в системе присутствует и зарегистрирована библиотека msvcr100.dll. Она необходима для запуска OpenOffice по завершении административной установки. Библиотека может быть загружена с сайта Microsoft по адресу http://www.microsoft.com/en-us/download/details.aspx?id=5555

Примечание: для выполнения процесса установки необходимы права администратора.

Убедитесь, что во временном каталоге системы достаточно свободного пространства, а также в наличии прав на чтение, запись и исполнение в нём. Перед началом установки закройте все остальные программы.

Проблемы при запуске программы

Трудности при запуске OpenOffice (например, зависание приложений), а также проблемы с экранным отображением часто вызваны драйвером видеокарты. При возникновении этих проблем обновите драйвер или используйте драйвер, включенный в состав используемой операционной системы. Для решения проблем при отображении трёхмерных объектов следует отключить параметр «Использовать OpenGL» в меню «Сервис - Параметры - OpenOffice - Вид - Графический вывод».

Сенсорные панели ноутбуков ALPS/Synaptics в Windows

В связи с проблемой драйвера Windows вы не сможете прокручивать документы OpenOffice, водя пальцем по сенсорной панели ALPS/Synaptics.

Чтобы включить прокрутку с помощью сенсорных панелей, добавьте следующие строки в файл конфигурации "C:\Program Files\Synaptics\SynTP\SynTPEnh.ini" и перезагрузите компьютер.

[OpenOffice]

FC = "SALFRAME"

SF = 0x10000000

SF |= 0x00004000

Замечание: расположение конфигурационного файла может отличаться в разных версиях Windows.

Комбинации клавиш

Только комбинации клавиш, не используемые операционной системой, могут быть задействованы в OpenOffice. Если комбинации клавиш в OpenOffice не действуют, как это описано в справке, проверьте не используются ли они уже операционной системой. Для разрешения подобных конфликтов Вы можете изменить комбинации, используемые операционной системой. Вы можете также изменить почти все комбинации для OpenOffice. За дополнительной информацией по этому вопросу обращайтесь к справочному руководству OpenOffice или документации операционной системы.

Проблемы при отправке документов как электронных писем из OpenOffice

При отправке документа посредством 'Файл - Отправить - Документ электронной почтой' или 'Электронной почтой как PDF' возможно аварийное завершение или «зависание» программы. Это происходит из-за входящего в Windows системного файла «Mapi» (Messaging Application Programming Interface), некоторые версии которого вызывают проблемы. К сожалению, решение не может быть сведено к указанию правильного номера версии этого файла. Для дополнительной информации посетите http://www.microsoft.com и выполните поиск в Базе знаний Microsoft по «mapi dll».

Присоединиться к проекту

Даже имея небольшой опыт в разработке программных продуктов или написании кода, можно внести свой вклад в этот важный проект по разработке программного обеспечения с открытым исходным кодом. И это действительно так!

По адресу http://openoffice.apache.org/get-involved.html вы найдёте сведения о различных проектах: от проектов локализации, портирования и групповой работы до проектов по созданию кода собственно ядра. Те, кто не являются разработчиками, могут присоединиться к проектам по документации или маркетингу. В рамках маркетингового проекта OpenOffice применяются как революционные, так и традиционные коммерческие методы распространения программного обеспечения с открытым исходным кодом вне зависимости от языковых и культурных границ. Внести свой вклад можно, просто рассказывая знакомым об этом наборе офисных программ.

Вы можете помочь, присоединившись к маркетинговому списку рассылки marketing@openoffice.apache.org, где можете сообщить о точках общения с прессой, медиа, правительственными агентствами, консультатнтами, школами, группами пользователей Linux и разработчиками в вашей стране и местном сообществе.
